# Nur-Islami

Aplikasi ibadah harian lengkap dengan jadwal sholat, Al-Quran digital, arah kiblat, dan tahlil.

## Fitur

- Jadwal sholat harian dan pengingat
- Al-Quran digital dengan pencarian
- Penunjuk arah kiblat
- Kumpulan tahlil dan doa harian

## Memulai

- Pastikan Git terpasang dan akun GitHub aktif
- Gunakan SSH (disarankan) atau HTTPS sesuai preferensi
- Lihat instruksi “Push ke GitHub” di bawah untuk publikasi awal

## Kontribusi

- Diskusikan perubahan melalui Issue/PR
- Ikuti standar kode dan gaya penulisan yang konsisten
